# Project_1
BankSight: Transaction Intelligence Dashboard is created to visualize the various banking details and its actions performed. 
Dashboard is created using Streamlit app utilizing VS Code, Python, MySQL to analyze the banking transactions and details.
Various csv and json datasets are utilized to refer the data pertaining to details of accounts, customer, transactions, loans.
Streamlit supports to make the various transactions like deposition, withdrawal, updation and deletion of customer details.
Overall, this project summarizes the quick and effective visualization of bank details.
